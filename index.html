ODE Types Reference

Linear First Order

Form: y' + P(x)y = Q(x)

Method: Integrating Factor

Separable

Form: dy/dx = f(x)g(y)

Method: Separation of Variables

Bernoulli

Form: y' + P(x)y = Q(x)y^n

Method: Substitution to Linear

Homogeneous

Form: M(x,y)dx + N(x,y)dy = 0

Method: Substitution v = y/x

Exact

Form: M(x,y)dx + N(x,y)dy = 0

Where: ∂M/∂y = ∂N/∂x

2nd Order Linear

Form: a(x)y'' + b(x)y' + c(x)y = f(x)

Methods: Characteristic equation, Variation of Parameters
